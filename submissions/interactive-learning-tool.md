## 1. 它解决什么学习问题

很多人在学 AI 概念时容易「看懂了，但说不出来」—— 只看不练，以为自己懂了，实际上过两天就忘。

这个工具解决的问题是：**用主动回忆（active recall）代替被动阅读**，让学习者通过反复自测来巩固 10 个 AI 基础概念的理解。每道题都包含：
- 概念的一句话解释
- 一个具体例子
- 一个常见误区

不管是考前复习还是日常预习，跑一遍就能快速检测自己哪些概念还没吃透。

## 2. 用户如何与它交互

纯 CLI 交互，打开终端运行一行命令即可。

**交互流程：**

```
① 运行 python3 concept-quiz.py
② 选择出题模式（1-4）
③ 每题：看题 → 输入答案 → 看标准答案 → 自评对错
④ 10 题跑完（或按 q 退出）→ 显示成绩总结
```

全程键盘输入，无需注册、登录或网络。

## 4. 哪部分由 AI 生成，哪部分由你人工修改 / 验证

| 组成部分 | AI 生成程度 | 人工修改 / 验证 |
|---------|-----------|---------------|
| **概念知识库**（CONCEPTS 数组里的 10 条卡片） | AI 根据当天对话整理出初版 | ✅ 逐概念对话修正：Workflow vs Agent 区分、Tracing vs Rollback 边界、Guardrails 四层分类等全部经过人工审校 |
| **CLI 代码逻辑**（出题、交互、计分、模式切换） | AI 编写初版 Python 代码 | ✅ 人工审查代码逻辑无 bug，运行测试验证正常 |
| **题目难度/顺序** | AI 设计 4 种模式 | ✅ 人工确认模式设计合理（cover 了从概念名→解释、解释→概念、例子→概念三种认知路径） |
| **README / 说明文档** | AI 辅助整理 | ✅ 人工复核，确保不含敏感信息 |

## 5. 还有哪些限制或下一步改进

### 当前限制

| 限制 | 说明 |
|------|------|
| **题库固定** | 目前只有 10 个概念，不能动态添加 |
| **自评制** | 对错由用户自己判断（y/n），不是机器自动匹配 |
| **纯 CLI** | 没有 GUI 或网页端，对不熟悉终端的人不友好 |
| **无持久化** | 成绩不保存，每次跑都是从头开始 |

### 下一步改进方向

1. **动态题库**：从外部 YAML/MD 文件加载卡片，新增概念只需编辑文件不用改代码
2. **自动匹配**：用语义相似度（embedding 比较）自动判断用户答案与标准答案的匹配程度，去掉自评 y/n
3. **Web 界面**：用 Streamlit 或 Flask 做成网页版，降低使用门槛
4. **间隔重复**：集成 SM-2 算法，按遗忘曲线安排复习频率
5. **导出功能**：run 完可导出错题列表，针对薄弱概念重点复习
